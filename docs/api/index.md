# API Reference

Complete API reference for `react-native-audio-browser`.

## Core Features

- **[Player](/api/features/player/)** — Setup and configuration
- **[Playback](/api/features/playback/)** — Play, pause, seek, volume, rate
- **[Queue](/api/features/queue/)** — Track management and navigation
- **[Browser](/api/features/browser/)** — Content navigation and tabs

## Now Playing & Metadata

- **[Now Playing](/api/features/nowPlaying/)** — Current track info
- **[Metadata](/api/features/metadata/)** — Stream metadata (ICY, chapters)

## Platform Features

- **[Sleep Timer](/api/features/sleepTimer/)** — Timed playback stop
- **[Favorites](/api/features/favorites/)** — Track favoriting
- **[Rating](/api/features/rating/)** — Track ratings
- **[Equalizer](/api/features/equalizer/)** — Audio equalization

## System

- **[Errors](/api/features/errors/)** — Error handling
- **[Battery](/api/features/battery/)** — Battery optimization (Android)
- **[Network](/api/features/network/)** — Connectivity status
- **[Remote Controls](/api/features/remoteControls/)** — External control handling
